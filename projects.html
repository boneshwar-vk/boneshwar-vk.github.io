Built a fully on-device conversational voicebot with end-to-end response latency under 1.5 seconds, designed to run without a GPU or cloud connection. The system is a cascaded architecture chaining three independently optimized low-latency modules: speech-to-text (STT) for transcription, a compact large language model (LLM) for reasoning and response generation, and text-to-speech (TTS) for natural voice output. Each stage is selected and tuned for minimal inference time — quantized models, streaming and chunked processing, and pipelined execution so modules overlap rather than run strictly in sequence. The result deploys on commodity hardware such as laptop CPUs and mobile phones, keeping conversations private, offline-capable, and responsive.
